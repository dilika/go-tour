# weekly/

Revue hebdomadaire (samedi). Un fichier par semaine : `weekly/week-NN.md`.

## Format (3 lignes, pas plus)

```
# Semaine NN (AAAA-MM-JJ → AAAA-MM-JJ)

## Avancée
<en 1-2 phrases : chapitres couverts, PRs poussées, h totales>

## Blocage
<en 1 phrase : ce qui a coincé, ou "aucun">

## Décision
<en 1 phrase : ajustement pour la semaine suivante, ou "maintenir le cap">
```

Règle : si tu ne peux pas résumer en 3 lignes, tu n'as pas réfléchi à ta semaine.
